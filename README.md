# canENDR
This is the First PCB I have ever designed and I designed it using KiCAD 10.0

Its a PCB that is used as a terminator for FRC CAN lines. It uses a 120 ohm SMD resistor and fits within a 1"x.75" profile making it small enough to fit in any bot. It was made using KiCAD 10.0 and gerber files are included along with a BOM 

All parts can be bought from DigiKey and part numbers are listed in the BOM along with links (the BOM is incomplete i neget around to seeing how much it would cost to make a batch of these PCBs but the cost might change depending on the year you are reading this).
